# About
This is a `React18 + TypeScript + Vite` project.

The plugin is as follows:
1. `Axios`
1. `ESLint`
2. `@reduxjs/toolkit`
3. `Redux`
4. `React Redux`
5. `PostCSS`
6. `SCSS`
7. `Ant Design`
8. `React Router Dom`
